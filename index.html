Summary: Latitude vs. X

The purpose of this project was to answer the question "What's the weather like as we approach the equator?" to determine how the weather changes as we approach the equator. To answer that question, various weather factors including temperature, humidity, cloudiness, and wind speed for more than 500 cities was captured using the OpenWeatherMap API.

The data was then imported into Python for analysis and subsequently visualized using Matplotlib. This site provides the source data as well as the visualizations that were generated from the analysis.
